# Tax advisor's page

This is simple page for tax advisor with blog and newsletter.

## Technologies
Project is created with:
* HTML 5
* CSS 3 
* SASS
* JavaScript
* JQuery
* Gulp

Page is responsive.

## Effects used on this website:
* responsive menu
* slide in on scroll 
* smooth scroll
* scroll back to top page arrow button
* typing
* expanding elements
